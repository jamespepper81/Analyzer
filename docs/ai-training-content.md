# BitSleuth - AI Bitcoin Wallet Analyzer

## Overview
BitSleuth is the most advanced AI-powered Bitcoin wallet analyzer and security tool available. It provides comprehensive analysis of Bitcoin wallets including security assessment, privacy analysis, transaction insights, and AI-powered recommendations.

## Key Features
- **AI-Powered Analysis**: Advanced machine learning algorithms analyze Bitcoin wallet patterns, behaviors, and security postures
- **Security Assessment**: Comprehensive security analysis including address reuse detection, dust attack identification, and opsec risk evaluation
- **Privacy Analysis**: Detailed privacy analysis to identify potential leaks and recommend best practices for Bitcoin privacy
- **Transaction Insights**: Deep dive into transaction history with visualizations, patterns, and performance analysis
- **Market Integration**: Real-time Bitcoin market data, mempool monitoring, and price analysis
- **Secure & Private**: Your private keys never leave your device. Only public blockchain data is analyzed

## Technology Stack
- **Frontend**: Next.js 16, React 19, TypeScript
- **AI/ML**: OpenAI GPT-4.1 Mini with Genkit framework
- **Blockchain Data**: Multiple Bitcoin data sources (Blockstream, mempool.space, blockchain.info)
- **UI**: Tailwind CSS 4, shadcn/ui components
- **Analytics**: Firebase (client-side only)

## Security & Privacy
- **No Private Keys**: Only analyzes public blockchain data using Bitcoin XPUB keys
- **Client-Side Processing**: All analysis happens in the user's browser
- **No Data Storage**: No user data is stored on servers
- **Open Source**: Transparent and auditable codebase

## How It Works
1. User enters their Bitcoin XPUB (Extended Public Key)
2. System fetches public blockchain data for all addresses derived from the XPUB
3. The platform reviews transaction patterns, security posture, and privacy practices
4. A comprehensive report is generated with insights and recommendations
5. Users can explore detailed analysis through various dashboard sections
